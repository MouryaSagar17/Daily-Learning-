# Daily-Learning-
Learning Java Daily Series

## Daily Progress

### Day 1:
- **EvenorOdd.java**: Checks if a number is even or odd.
  - *Logic*: `number % 2 == 0`
- **PostiAndNegi.java**: Determines if a number is positive or negative.
  - *Logic*: `number > 0`
- **input.java**: Basic user input handling.
  - *Logic*: `Scanner sc = new Scanner(System.in);`

### Day 2:
- **LeaporNot.java**: Checks for leap years.
  - *Logic*: `(year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)`
- **SumN.java**: Calculates the sum of N numbers.
  - *Logic*: Loop and accumulate.
- **SumofRangeN.java**: Sums numbers within a given range.
  - *Logic*: Loop from start to end and add.
- **greatestof2.java**: Finds the largest of two numbers.
  - *Logic*: `a > b`
- **greatestof3.java**: Finds the largest of three numbers.
  - *Logic*: `a > b && a > c`

### Day 3:
- **PrimeUptoN.java**: Finds all prime numbers up to N.
  - *Logic*: Nested loops to check for factors.
- **PrimeorNot.java**: Checks if a number is prime.
  - *Logic*: Check for factors from 2 to `sqrt(number)`.
- **ReverseaNumber.java**: Reverses a given integer.
  - *Logic*: `reverse = reverse * 10 + remainder`
- **SumofN.java**: Calculates the sum of N numbers.
  - *Logic*: Loop and accumulate.
- **palindrome.java**: Checks if a number is a palindrome.
  - *Logic*: Compare the number with its reverse.

### Day 4:
- **ArmStrong.java**: Checks for Armstrong numbers.
  - *Logic*: Sum of digits raised to the power of the number of digits.

### Day 5:
- **Factorial.java**: Calculates the factorial of a number.
  - *Logic*: `factorial = factorial * i`
- **Factors.java**: Finds all factors of a number.
  - *Logic*: Loop from 1 to the number and check for divisibility.
- **FibonacciSeries.java**: Generates the Fibonacci sequence.
  - *Logic*: `next = first + second`
- **Power.java**: Calculates the power of a number.
  - *Logic*: `Math.pow(base, exponent)`
- **PrimeFactor.java**: Finds the prime factors of a number.
  - *Logic*: Repeatedly divide by the smallest prime factor.
- **StrongNumber.java**: Checks for Strong numbers.
  - *Logic*: Sum of the factorials of the digits.
